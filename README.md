# 🌍 Wanderlust – Travel Booking Platform

Wanderlust is a modern travel booking platform that helps travelers discover amazing destinations, explore detailed travel information, and book unforgettable experiences with ease. The platform provides a seamless user experience with secure authentication, destination management, and booking functionality.

## 🚀 Live Demo

**Live Site:** https://wanderlust-ivory-seven.vercel.app

---

## ✨ Features

### 👤 Authentication

* Secure user registration and login
* Session-based authentication
* Protected routes for authenticated users
* User profile management

### 🏝️ Destination Management

* Browse all available destinations
* View detailed destination information
* Featured destinations section
* Add new destinations (authorized users)
* Beautiful destination cards with images and pricing

### 📅 Booking System

* Book travel destinations easily
* View personal bookings
* Manage booked trips
* Booking confirmation workflow

### 🎨 Modern User Interface

* Fully responsive design
* Mobile-friendly navigation
* Interactive destination cards
* Custom 404 and error pages
* Clean and modern travel-themed layout

### ⚡ Performance

* Server-side rendering with Next.js
* Incremental Static Regeneration (ISR)
* Optimized image loading
* Fast page navigation

## 🔧 Installation

### Clone Repository

```bash
git clone <repository-url>
cd wanderlust
```

### Install Dependencies

```bash
npm install
```

### Environment Variables

Create a `.env.local` file:

```env
NEXT_PUBLIC_SERVER_URL=your_backend_url
```

### Run Development Server

```bash
npm run dev
```

Visit:

```bash
http://localhost:3000
```
